### System logowania we Vue z wykorzystaniem serwera node oauth2

## Użyty serwer : https://www.npmjs.com/package/node-oauth2-server-example

## Technologie użyte:

# -Vue 3
# -Bootstrap
# -Vuex
# -node
# -axios

## Web Setup:

# cd sys-login
# npm install
# npm run serve

## Server setup:

# npm start
